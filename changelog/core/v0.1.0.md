## Features
- desktop-app에서 DB 로직 추출하여 독립 패키지로 분리
  - connection, repositories, migrations를 injectable DB path로 재설계
- SQLite 연결 관리 (WAL 모드, busy_timeout 5000ms)
  - 다중 프로세스 동시 접근 지원 (desktop-app + netior-mcp)
- 8개 migration 파일 (001~008)
  - initial, modules/hierarchical-canvas, archetypes, concept-content, app-settings, canvas-1n-and-types, edge-visual-overrides, canvas-layout
- Repository 패턴 구현 (archetype, canvas-type, canvas, concept-file, concept-property, concept, editor-prefs, relation-type, settings, module, project)
  - CRUD, 검색, cascade 삭제, boolean 변환, 기본값 처리
- 프로젝트 root_dir 마이그레이션 지원
  - 경로 미발견 시 모달 프롬프트, 모듈 디렉토리 일괄 업데이트

## Internal
- moc-core -> netior-core 리네이밍
- 테스트 하니스 (인메모리 SQLite, 64개 테스트 케이스)
- tsup 빌드 설정 (ESM + CJS 이중 출력)
