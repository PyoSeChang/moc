## Features
- `files` 테이블 신규 — 파일/디렉토리를 1급 엔티티로 관리 (id, project_id, path, type, metadata)
- `FileRepository` CRUD (create, get, getByPath, getByProject, update, delete)
- `canvas_nodes.metadata` 컬럼 — 노드별 JSON 메타데이터 저장
- `canvas_nodes.file_id` FK — files 테이블 참조
- `getCanvasFull()` — files LEFT JOIN으로 파일 데이터 반환
- `updateCanvasNode()` — metadata 업데이트 지원
- Migration 009: 기존 file_path/dir_path 데이터 자동 이관

## Breaking Changes
- `canvas_nodes.file_path`, `canvas_nodes.dir_path` 컬럼 제거 → `file_id`로 대체
- `addCanvasNode()` — `file_path`/`dir_path` 파라미터 제거, `file_id`/`metadata` 추가
- `concept_files` 테이블 DROP
- `ConceptFileRepository` (createConceptFile, getConceptFilesByConcept, deleteConceptFile) 제거
