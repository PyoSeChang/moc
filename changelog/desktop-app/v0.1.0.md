## Features
- 프로젝트 홈 화면, Zustand 스토어, UI 컴포넌트 시스템 구축
  - Button, Input, Select, Modal 등 16종 공용 컴포넌트
- 캔버스 엔진 직접 구현 (CSS transform + SVG)
  - 노드 렌더링, pan/zoom, 엣지, dot grid 배경
- 사이드바, 파일 트리, 확장자 기반 에디터 독 시스템
  - `.md` → MarkdownEditor, `.png/.jpg` → ImageViewer 등 자동 선택
- 설정 모달, 키보드 단축키, 개념 검색
- 모듈 시스템, 계층적 캔버스, 멀티 모드 에디터, 개념 생성 UX
- Activity bar 리본으로 사이드바 전환
- 에디터 셸 개선: 분할 뷰, 탭 드래그앤드롭, 분리 모드
- Archetype 시스템으로 구조화된 개념 정의
  - IconSelector, TypeSelector, ArchetypeEditor UI
- Monaco 코드 에디터, PDF 뷰어, 이미지 뷰어 통합
- 터미널 에디터 (node-pty + xterm.js)
- 개념 콘텐츠 시스템 (contentEditable 에디터)
- VS Code 스타일 분할 뷰 탭 스트립
- 디렉토리 탭 (vscode-icons, 탐색기 기능)
- 브라우저 스타일 에디터 탭, 통합 배경
- Discord 스타일 설정 패널
- 탭 스트립 컨텍스트 메뉴
- FileTree 개선: dotfiles, 컨텍스트 메뉴, i18n, 에러 토스트
- UX 개선: 마지막 프로젝트 복원, 윈도우 위치 저장, dev/prod DB 분리
- Canvas 1:N, RelationType, CanvasType, edge/node 확장
- Edge CRUD, 캔버스 트리, CanvasEditor, 브레드크럼, 시각적 오버라이드
- Obsidian 스타일 라이브 프리뷰 마크다운 에디터 (CodeMirror 6)
- 디렉토리 패널: 자동 모듈 초기화, 지연 로딩, 스피너
- 캔버스 레이아웃 플러그인 시스템 + 수평 타임라인
- Gantt 차트: 드래그 컨트롤, 색상 매핑, 레인 순서, 아키타입 제한
- useEditorSession 프레임워크 (스냅샷 기반 dirty tracking)
- 프로젝트별 상태 격리, 터미널 닫기 경고
- PDF 뷰어 blob URL + URL.parse 폴리필
- Narre AI 어시스턴트 MVP: 챗 UI, SSE 스트리밍, 도구 실행 로그
  - agent-server → Claude Agent SDK 전환
  - 네이티브 모듈 충돌 해결 (@moc/core 분리)
- 터미널 런타임을 VS Code 기반 서비스 경로로 교체
- 테스트 하니스: Vitest (shared + desktop-app 유닛/스토어 테스트)
