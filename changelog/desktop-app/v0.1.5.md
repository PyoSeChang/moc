## Features
- YAML frontmatter 라이브 프리뷰 위젯 추가
  - 옵시디언 스타일 속성 테이블 렌더링 (key-value)
  - 멀티라인 YAML 블록 스칼라 지원 (`>`, `|`)
  - 커서 진입 시 raw 소스 표시, 이탈 시 위젯 표시
  - frontmatter 영역 내 HorizontalRule 노드 렌더링 방지
