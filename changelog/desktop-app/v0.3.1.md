## Features
- 디렉터리 패널 폴더 우클릭 메뉴에 "터미널에서 열기" 동작을 추가해 해당 폴더 기준으로 터미널 탭을 바로 열 수 있도록 개선

## Fixes
- 터미널 탭 전환 시 이전 탭의 cwd가 재사용되던 문제를 수정해 폴더별 터미널 시작 경로가 섞이지 않도록 변경
- 첨부 파일 경로 검증 시 절대경로를 프로젝트 루트에 다시 붙여 누락 파일 모달이 잘못 뜨던 문제를 수정
- 절대경로/상대경로 파일 엔티티 검증 케이스를 테스트에 추가
