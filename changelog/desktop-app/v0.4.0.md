## Features
- PDF 뷰어: fit-to-width 기본 줌, 두 쪽 보기, TOC 사이드바 (file metadata 연동), 페이지 입력 너비 수정
- PDF 뷰어: 보기 설정 (page, scale, viewMode) file metadata에 영구 저장/복원
- EdgePanel: 가장자리 슬라이드인 패널 공통 UI 컴포넌트 (trapezoid hint, hover trigger, pin 지원)
- MarkdownToc: 계층형 트리 (collapse/expand), 커서 기반 active heading 하이라이트, 기본 2레벨 펼침
- useViewState: 탭별 인메모리 뷰 상태 훅 (탭 전환 시 커서/스크롤/페이지 보존)

## Internal
- TerminalTodoPanel을 EdgePanel로 리팩터링 (hover/pin 로직 공통화)
- MarkdownToc를 EdgePanel로 리팩터링 (margin 모드 제거, EdgePanel 통일)
- CodeEditor에 tabId prop 추가, useViewState로 커서/스크롤 보존
- MarkdownEditor에 useViewState로 커서/스크롤 보존
- FileEditor에 fileId 해석 로직 추가 (절대경로 매칭 fallback)
- editor-store closeTab에서 clearViewState 호출
