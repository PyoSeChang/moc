## Features
- detached editor window를 host-aware editor workspace로 재구성해 멀티 탭, 컨텍스트 메뉴 이동, 창 간 탭 이동 흐름을 지원
- 메인 워크스페이스에 detached 탭을 다시 붙일 때 Side/Float 드롭 존을 분리해 메인 레이아웃으로 명시적으로 복귀 가능

## Fixes
- detached window bootstrap race를 막아 빈 state hydrate 후 즉시 닫히는 문제 수정
- cross-window tab drag/drop이 Electron window 경계를 넘어갈 때 payload를 잃거나 금지 커서가 뜨던 문제 수정
- detached terminal 탭이 검은 화면으로 남던 문제를 고쳐 cwd bootstrap, 세션 replay, PTY 재부착을 복구
- host별 active tab, focused host, strip context menu 대상 host 해석이 어긋나던 문제 수정

## Internal
- editor state bridge에 host bootstrap validation과 workspace context hydration 로직 추가
- terminal open 경로를 공용 helper로 통일하고 PTY replay buffer를 main process에서 관리
