## Features
- Narre `/index` 워크플로우: 슬래시 커맨드 인터셉트 -> PdfTocInputForm (페이지 범위 입력) -> JSON toc_params 전송 -> 승인 후 저장
- 파일 멘션 시스템: NarreMentionPicker에 file 카테고리 추가, NARRE_SEARCH_MENTIONS에 file entity 검색, mention chip에 mentionPath 저장
- narre-server-manager: `ELECTRON_RUN_AS_NODE` + `process.execPath`로 MCP 서브프로세스 spawn -> better-sqlite3 네이티브 모듈 호환
- pdf-toc-utils에 `buildIndexMessage()`, `TocParams` 타입 추가
