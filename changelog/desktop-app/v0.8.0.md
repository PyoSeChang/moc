## Features
- 마크다운 목차(TOC) 토글 단축키 추가 (`Ctrl+Shift+O`)
- 사이드/전체 에디터 모드 전환 단축키 추가 (`Ctrl+\`)
- EditorViewModeSwitch를 탭 레벨(float/detached)과 패인 레벨(side↔full, minimize) 버튼 그룹으로 분리

## Fixes
- 분할화면 후 드래그 존이 잔상으로 남는 버그 수정 (flushSync 적용)
- 마크다운 목차 pinned 상태가 탭 전환 시 소실되는 버그 수정 (useViewState로 탭별 저장)
- Hook server 포트를 동적 파일로 관리하여 앱 재시작 시 stale port 문제 해결
