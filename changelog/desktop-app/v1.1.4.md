## Features
- `netior-service`를 Node 22 sidecar 기준으로 정리하고 desktop-app을 thin client 경계로 맞췄다.
- Narre provider UI bridge와 schema를 공통화해 Claude/OpenAI/Codex 경로 드리프트를 줄였다.
- desktop Settings와 runtime 문서를 현재 service-first 구조에 맞게 갱신했다.

## Fixes
- `better-sqlite3` 소유권을 `netior-service`로 모아 Electron/Node ABI 충돌을 줄였다.
- `narre-eval`, `netior-mcp`, `narre-server`가 DB 직접 접근 대신 service 경유로 동작하게 정리했다.

## Internal
- `desktop-app`용 bundled Node runtime 패키징 경로를 유지한 채 sidecar 런타임 분기를 단순화했다.
- dead Narre runtime code와 legacy UI helper를 shim/shared layer로 정리했다.
