## Fixes
- Unpacked `@netior/service`, `@netior/narre-server`, and `@netior/mcp` for the packaged Windows app so bundled Node sidecars no longer try to execute files from `app.asar`.
- Updated sidecar path resolution to prefer `app.asar.unpacked` paths in production.
- Added a startup timeout fallback for persisted window bounds loading so the main window is not blocked by config RPC delays.
- Added `desktop-main.log` file logging for packaged main-process startup diagnostics.

## Internal
- Kept Narre MCP path resolution aligned with the packaged `@netior/mcp` layout.
