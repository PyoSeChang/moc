## Features
- 시나리오 기반 e2e 테스트 인프라 구현
  - YAML 시나리오 로더, HTTP+SSE 러너, DB assertion + LLM judge 채점기
- CLI 도구: 시나리오 실행 및 결과 리포트 생성
  - TSV 요약 + 트랜스크립트 리포터
- MVP 시나리오 3종: init-project, type-update, cascade-delete
- 테스트 하네스: DB 초기화 + 프로세스 관리 (narre-server, mcp 자동 기동/종료)
