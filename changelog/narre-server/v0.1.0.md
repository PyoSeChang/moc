## Features
- Express HTTP 서버 + SSE 스트리밍 기반 AI 에이전트 구현
  - 파일 기반 세션 저장 및 resume 지원
- Claude Agent SDK로 전환, moc-mcp를 stdio MCP 서버로 연결
  - 수동 stream/tool 처리 제거, SDK 에이전틱 루프 사용
- 슬래시 커맨드 시스템: command-router, `/command` 엔드포인트
  - `/onboarding`: 4단계 타입 시스템 빌더 (Archetype -> RelationType -> CanvasType -> Concept)
- UI 도구 (propose/ask/confirm): createSdkMcpServer로 응답 카드 생성
  - ProposalCard, PermissionCard, InterviewCard, SummaryCard 지원
- 시스템 프롬프트 메타데이터를 desktop-app에서 `/chat` body로 수신
  - DB 직접 접근 제거, 모든 DB 작업은 moc-mcp 경유

## Fixes
- SDK 메시지 re-yield 중복 방지 가드 추가
  - Claude Agent SDK가 동일 메시지를 재전달하는 현상 대응
- better-sqlite3 네이티브 모듈 NODE_MODULE_VERSION 충돌 해결
  - agent-server에서 @moc/core 의존성 제거로 Electron/Node.js 바이너리 분리
