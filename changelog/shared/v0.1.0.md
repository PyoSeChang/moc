## Features
- 프로젝트 기본 타입 정의 (Project, Concept, Canvas, CanvasNode, Edge, ConceptFile)
  - 초기 모노레포 스캐폴딩 시 전체 데이터 모델 설계
- IPC 채널 상수 및 기본값 상수 체계
  - 모든 main-renderer 통신의 단일 진실 공급원
- i18n 시스템 (ko/en 이중 언어 지원)
  - translate 함수, 키 검증 포함
- Module 시스템 타입 및 계층적 캔버스 타입 추가
  - 모듈, 디렉토리 패널, 계층 탐색 지원
- Archetype 시스템 타입 (name, icon, color, node_shape, file_template, fields)
  - 구조화된 Concept 정의를 위한 타입 클래스
- 에디터 쉘 타입 (split view, tab drag-drop, detached mode)
  - VS Code 스타일 per-pane 탭 스트립 지원
- Terminal 에디터 타입 및 IPC 채널
  - node-pty + xterm.js 통합용 상수
- Canvas 1:N, RelationType, CanvasType, Edge/Node 확장 타입
  - Concept:Canvas 일대다 관계 및 타입 시스템 도입
- Edge CRUD, CanvasEditor, breadcrumb 관련 타입 추가
  - 캔버스 인터랙션 UI 지원
- 마크다운 에디터 타입 및 i18n 키
  - Obsidian 스타일 라이브 프리뷰 에디터 지원
- 캔버스 레이아웃 플러그인 타입 (horizontal timeline 등)
  - Gantt 차트 및 타임라인 레이아웃 지원
- Narre AI 어시스턴트 타입 (NarreSession, NarreMessage, NarreMention 등)
  - 9개 IPC 채널, narre 네임스페이스 i18n 키
- 슬래시 커맨드 시스템 타입 및 상수
  - 응답 카드, 온보딩 커맨드 지원
- Claude Code 연동 타입 (세션 감지, 상태 표시)
  - hook 서버, 세션 추적용 타입
- VS Code 기반 터미널 서비스 타입
  - 터미널 런타임 교체에 따른 상수/타입 재설계
