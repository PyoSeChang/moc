# EDITOR

* 분할화면 시에 탭 상단 border 날아가는 문제
* 드래그 해서 화면 분할할 떄, side Editor로 여는 드랍 존이 사라지지 않음
* 탭 스트립이 가득 찼을 때, dirty 표시가 뜨면 탭의 오른쪽 border가 안 보임
* network object editor에서 새로 생성 시에는  바로 dirty 표시가 떠야함
* network object editor에 properties 수정 같은 부분에서 변경 되어도 dirty 표시가 안 뜸
* 다른 pane으로 focus가 넘어갈 때, 기존 탭의 activate tab 넘버가 바뀜 파일을 열 때 그러는 것 같음
